Emoji-Aware Sentiment Analysis Using Dual-Stream Neural Networks
April 2025
Developed an emoji-enhanced sentiment analysis system using a dual-stream neural architecture that processes text and emoji inputs independently before merging them via a cross-modal attention mechanism. The project combined fine-tuned BERT embeddings for text and emoji2vec embeddings for emojis, resulting in more accurate interpretations of social media sentiment, especially in sarcasm-heavy or emoji-rich posts. Evaluated on Sentiment140 and Emoji Sentiment Ranking datasets, this approach significantly outperformed traditional text-only models in sentiment classification tasks.
ModELS Controller – Modularly Enabled LED Sequencing
March 2025 – April 2025
Designed and implemented the ModELS Controller, a modular LED sequencing peripheral for the SCOMP processor, using VHDL. This hardware controller interprets bitstring instructions to control LED brightness, pattern frequency, and selection via parallel state machines. The design supports 13 customizable LED patterns and enables dynamic behavior like modulation, inversion, and individual LED toggling. By focusing on efficiency, scalability, and user-friendly bit encoding, our modular architecture delivers a robust, flexible system for interactive LED-based applications.
